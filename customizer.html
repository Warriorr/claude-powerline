        📦 How to install
        ▶
      

      

        
1. Install the package

        
npx -y @owloops/claude-powerline@latest

        
2. Save your config

        
Copy the generated JSON above and save it to:

        
~/.claude/claude-powerline.json

        
3. Add to your Claude hooks

        
In ~/.claude/settings.json, add under hooks:

        
"PreToolUse": [{ "hooks": [{ "type": "command", "command": "npx @owloops/claude-powerline" }] }]

        
4. (Optional) Install Nerd Fonts

        
For icon support (clock/calendar indicators), install a Nerd Font and set your terminal to use it:

        nerdfonts.com/font-downloads
